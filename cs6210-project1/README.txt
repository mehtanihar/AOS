This is an implementation of Credit based Scheduler for Assignment 1 
of Advanced Operating Systems Course at Georgia Institute of Technology.

In order to test the code, please navigate to the directory containing the Makefile.
1. Use make to compile the GTThreads library
2. Use make matrix to compile the gt_matrix.c
3. Finally use ./bin/matrix 0 for priority scheduling or ./bin/matrix 1 for credit-based scheduling.
4. A result.txt file is provided for visualizing the output of the code.
5. To reproduce the results, please use ./bin/matrix 1 &> output.txt
